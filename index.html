Case Study 1: Mobile App Sound Design

The Challenge

Creating a cohesive sonic identity for Hinge Health's mobile application that enhances user engagement while providing clear feedback for interactions across a platform serving 70,000+ daily users.

My Approach

  • Conducted sonic audits of existing health applications to identify gaps and opportunities
  • Collaborated with UX designers to map critical user journeys and identify key interaction points
  • Developed a sound palette that aligned with brand values while ensuring accessibility across diverse user groups
  • Created a modular sound system that could scale across different interaction types

Implementation

  • Built custom implementation pipelines using Wwise integrated with Unity
  • Developed adaptive sound behaviors that responded to user interaction patterns
  • Created lightweight audio assets optimized for mobile performance
  • Implemented A/B testing framework to measure sound effectiveness

Results

  • 18% increase in user session duration after implementing new sonic identity
  • Reduced user errors through clearer audio feedback at critical interaction points
  • Established scalable audio guidelines that engineering teams could implement independently
  • Positive user feedback specifically mentioning the "satisfying" nature of interactions

Case Study 2: Interactive Audio Experience

The Challenge

Designing an immersive audio experience for a spatial computing prototype that needed to provide intuitive feedback without relying on visual cues, enhancing the feeling of presence while minimizing cognitive load.

My Approach

  • Researched psychoacoustic principles to understand how sound creates spatial awareness
  • Developed a sonic hierarchy that distinguished between primary and secondary interactions
  • Created a testing methodology to evaluate effectiveness of audio cues in eyes-free scenarios
  • Collaborated with haptic designers to create multi-sensory feedback patterns

Implementation

  • Utilized spatial audio techniques to create directional awareness cues
  • Implemented adaptive systems that adjusted based on environmental conditions
  • Created a procedural audio system using Pure Data that generated sounds in real-time
  • Built custom testing tools to evaluate user response to different audio feedback methods

Results

  • Users reported 40% higher confidence in system interactions compared to visual-only feedback
  • Reduced learning curve for new users through intuitive audio mapping
  • Created a framework for audio-haptic integration that became a reference for future products
  • Successfully implemented in prototype that later influenced production features
